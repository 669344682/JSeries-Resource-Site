# CommunityMTA


## Running Locally

To run Locally you'll need Meteor.js

    Clone this Git Repo

    git clone https://github.com/codylewiz/CommunityMTA.git
    
    CD into the project

    cd ./CommunityMTA
      
    Install the required NPM packages

    meteor npm install --save jimp imagemin decay

    Start Meteor

    meteor --port (Port here)
    
    
Not tested in linux
